## Recal Social WebSocket

The WebSocket for handling real-time communication in the Recal Social chat application, written with [Nest](https://github.com/nestjs/nest).
This project uses the `pnpm` package manager.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
